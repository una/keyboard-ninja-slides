# Step 3: Navigating Files - multiple cursors: `cmd` - also, `cmd`+ `drag` - highlight line: `cmd` + `l` - find all of a query: `cmd` + `d` - duplicate line: `cmd` + `shift` + `d` - tab navigation: `ctrl` + `tab` - wrap element: `ctrl` + `w` + `wrap` - wrap element adv: `a[href="http://link.com"]`
# Step 4: Navigating Panels - open/close sidebar: `cmd` + `k` + `b` - sidebar navigation: `ctrl` + `0` - tab navigation: `ctrl` + `tab` - fuzzy search: `cmd` + `p`
